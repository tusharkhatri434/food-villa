## Run Locally

Clone the project

```bash
  git clone https://github.com/tusharkhatri434/food-villa.git
```

Go to the project directory

```bash
  cd Food-villa
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Running Tests

To run tests, run the following command

```bash
  npm run test
```
